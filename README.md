Cat Clicker
===============================

## How to Play

You can play Cat Clicker [HERE](https://tahkus.github.io/cat-clicker/).

## Installation

* ``$ git clone https://github.com/Tahkus/cat-clicker.git``
* Open index.html in your web browser in order to play the game.
